# 🚀 COSMIC Clipboard Pro: Universal RAM Edition

**The only clipboard engine that puts privacy, comfort, and flexibility first.**

If you've used Linux on Wayland, you know the pain: You copy a URL or a photo, close the app, and when you go to paste it... **it's gone.** 

**COSMIC Clipboard Pro** solves this forever. It provides **Silent Persistence** and a **Windows-Style (`Win + V`) History** for everything you copy.

---

## 🌟 The "Universal" Power 🖼️
Most clipboard scripts only handle text. **We handle it all.**
- **Text & Links:** Standard copy-paste works perfectly.
- **Images (PNG, JPG):** Copy an image from your browser or folder, close the app, and still paste it into your editor.
- **GIFs:** Yes, it even remembers and persists your favorite GIFs!

## 🧠 The RAM-Only Revolution 
Your data stays in your RAM, never on your disk.
- **Volatile Storage:** History is stored in **System RAM** (`/run/user/`).
- **Total Wipe on Reboot:** Your history is physically erased every time you restart.
- **Zero Disk Trace:** Your secret notes and passwords never touch your hard drive. It's the ultimate "Invisible Security."

## ⚡ Memory Efficiency: A Lightweight Ghost
- **Low Footprint:** Uses less than **10MB** of RAM.
- **Automatic Cleanup:** Deletes the oldest items once you hit the limit.
- **Instant Speed:** RAM storage means searching is lightning fast.

---

## 🚀 Key Features
- **Silent Persistence:** Copy something, close the app, and it **stays** in your clipboard.
- **Windows-Style History (`Win + V`):** A beautiful, searchable list of your previous copies.
- **Smart Auto-Paste:** Select an item (text, image, or GIF) and it automatically pastes.
- **Zero Setup:** The installer **automatically configures your Super + V shortcut** for you.

---

## 🚀 Installation
1. **Clone it:**
   ```bash
   git clone https://github.com/kernel-ux/cosmic-clipboard-pro.git
   cd cosmic-clipboard-pro
   ```
2. **Run the Universal Installer:**
   ```bash
   chmod +x install.sh
   pkexec ./install.sh
   ```
3. **Restart & Enjoy.** (Super+V is already set for you!)
